Markdown

Hexo

Markdown是一种可以通过简单标记语法编辑，使普通文本呈现特定排版格式的标记语言。

  • 宗旨：「易读易写」
  • 功能：排版、图片、表格、脚注、内嵌HTML等
  • 特点： 易读易写、兼容HTML、可代码式完全键盘输入

Markdown语法

1. 排版

1.1 字体

Markdown 输出 快捷键
# Heading H1
## Heading H2 Ctrl/⌘ + H
**Bold** Bold Ctrl/⌘ + B
*Bold+Emphasize* Bold Ctrl/⌘ + B
*Emphasize* Emphasize Ctrl/⌘ + I
~~Strike-through ~~ text Ctrl + Alt + U

1.2 格式

Markdown 输出 Markdown 输出
> Enter换行 ` ` 内嵌代码
三个* 分割线 Tab 代码块
三个空格 引用 ` 代码

1.3 首行缩进

  • &+nbsp; (无+号)
  • &+#160; (无+号)
  • 汉字全角输入两个空格

1.4 表格

1.5 链接

  • 跳转链接：[百度](www.baidu.com)
  • 插入图片：![图片]（地址）

  • [我的主页](wwww.best5.club) –
    我的主页
  • 插入图片的[]框中为图片属性，相当于img的 alt 属性。
    比如：![post-cover]（地址） //部分主题中post-cover 为背景图片
